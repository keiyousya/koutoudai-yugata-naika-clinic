# 会計業務

## 使用機器について

| 機器 | レセコン |
| --- | --- |
| 受付用デスクトップPC | 使用可 |
| 処置室ノートPC | 使用不可 |

- **レセコンは受付用デスクトップPCからのみ操作可能**
- 処置室ノートPCからはレセコンを直接開けない

## 基本の流れ

1. レセコンで医師が入力した診療行為を登録し、会計金額を患者にお伝えする
2. 決済方法を確認し、レセコンに入力する
3. レセコンで会計を確定し、領収証明細書を印刷する
4. 領収証明細書を提示し、改めて金額をお伝えする
5. それぞれの決済方法に従って決済を行っていただく

## 現金決済

> 当院は電子レジ端末を導入していないため、手動計算ではミスが発生しやすい。患者さんへの不利益を防ぐため、**必ず電卓を使用**すること。

1. レセコンで診療行為を登録し、会計金額をお伝えする
2. 決済方法を確認し、「現金」でレセコンに入力する
3. レセコンで会計を確定し、**領収証明細書**を印刷する
4. **会計額を電卓に打ち込み、患者さんに提示する**
5. 現金を受け取り、**電卓でお預かり金額から会計額を引く**
6. **お釣り額を電卓で患者さんに提示し、お釣りをお渡しする**

## 対面キャッシュレス決済

1. レセコンで診療行為を登録し、会計金額をお伝えする
2. 決済方法を確認し、「キャッシュレス」でレセコンに入力する
3. レセコンで会計を確定し、**領収証明細書**を印刷する
4. 領収証明細書を提示し、改めて金額をお伝えする
5. キャッシュレス端末で支払いを完了する
6. キャッシュレス端末から発行される**売上票**と**領収証明細書**を合わせてお渡しする

## LINEクレジットカード決済

1. レセコンで診療行為を登録し、会計金額をお伝えする
2. 決済方法を確認し、「LINEクレカ決済」でレセコンに入力する
3. レセコンで会計を確定し、**領収証明細書**を印刷する
4. 領収証明細書を提示し、改めて金額をお伝えする
5. カルテから「LINEクレカ決済」を実行し、支払い完了を確認する
6. **領収証明細書**をお渡しする

> **注意**: エラーで支払いが完了しない場合は、対面での支払い（現金またはキャッシュレス）をお願いする。

## 処方箋のお渡し

- 処方箋がある場合は会計時にお渡しする
- 当院は原則**院外処方**（近隣の調剤薬局で受け取り）
- 近隣薬局が閉まっている時間帯に限り、急性疾患の一部薬剤は院内でお渡しする場合がある
- 慢性疾患の薬（降圧薬・糖尿病治療薬等）は必ず院外処方とする

> **重要**: 処方箋の有効期限は**4日間**。期限が切れると無効になるため、お早めに薬局で受け取るようお伝えする。

### 調剤薬局のご案内

受付に**周辺調剤薬局の一覧とマップ**（平日・土曜・日曜別）のラミネートを用意しています。患者さんにお見せしながら、現在空いている薬局をご案内してください。

**注意点**:
- 「〇〇薬局に行ってください」と**特定の薬局を指定することは法律で禁止**されています（医薬品医療機器等法・療養担当規則）
- あくまで「近くだと、この時間は〇〇薬局と△△薬局が開いています」のように**選択肢を提示する**にとどめてください

### かかりつけ薬局について

> 毎回全員に確認する必要はない。複数の医療機関にかかっている患者さんには、近隣薬局を案内しつつ「かかりつけの薬局があればそちらでも大丈夫です」程度に伝える。

**かかりつけ薬局のメリット**（患者さんに聞かれた場合の説明用）:
- 複数の医療機関の処方薬を一元管理し、**飲み合わせ（相互作用）をチェック**してもらえる
- **薬の重複**を防げる
- 体質やアレルギー歴を把握した上で、継続的な**服薬指導**を受けられる

## 領収証の再発行

- **原則**: 領収証の再発行は不可
- **再発行が必要な場合**: 1枚につき実費**550円**で発行する
- **年間の支払い証明書**: 1通**550円**で発行可能

> 領収証は大切に保管いただくようお伝えしましょう。
