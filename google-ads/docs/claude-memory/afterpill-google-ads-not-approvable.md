勾当台夕方内科クリニックの「アフターピル」キャンペーン(campaign_id `<AFTERPILL_CAMPAIGN_ID>` / 広告グループ `<AFTERPILL_AD_GROUP_ID>` / 広告 `<AFTERPILL_AD_ID>`)は、Google広告ポリシー `PRESCRIPTION_DRUG_SALE`(FULLY_LIMITED)＋`BIRTH_CONTROL`(LIMITED)で不承認。2026-07-01に対策を実施:

- LP `/afterpill/` から具体的な薬剤名「レボノルゲストレル」を全除去し診療表現へ一般化、価格カードを「緊急避妊のご相談・診療(自費診療)」に変更、医療機関情報ブロック追加
- キーワード「アフターピル 処方 仙台」を削除
- ポリシーマネージャーから「ポリシーに沿って変更を実施した」で再審査請求

→ 2026-07-03 時点で**再審査「失敗」で確定**。認定ルートも不可(対面診療のみの医療機関はオンライン薬局/遠隔医療に該当せず、Googleサポートも「申請しても承認されない」と回答)。**アフターピルキャンペーンはREMOVEDで削除済み**(残る有効キャンペーンは「2026年06月リスティング広告」のみ)。

**LPの現状(2026-07-03):** 広告を断念したため、広告対応で除去していた薬剤名「レボノルゲストレル」はLPに復活済み(SEO優先で元に戻した)。ただし広告対応で追加した**医療機関情報ブロック**と「対面診察が必要・薬のみの販売なし」注記は有用なため残してある。つまりLPは薬剤名あり＋医療機関情報ありの状態。

**Why:** 日本では処方薬関連用語(アフターピル/緊急避妊薬/処方 等のカテゴリ名自体)を一般向け広告文・LP・キーワードで使うことが原則禁止で、薬剤名を消しても回避できない。構造的に検索広告は出せない。

**How to apply:** アフターピルの検索広告掲載を再度試みない。集患は `/afterpill/` のSEO(構造化データ・FAQ構築済み)と、承認済みで無期限配信中の「2026年06月リスティング広告」(一般クリニック広告)に集中する。Google広告CLIは [[google-ads-cli-gads]] の `gads` を使用。
