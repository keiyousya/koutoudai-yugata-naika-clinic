# 在庫管理

## 管理対象

| 分類 | 例 |
|-----|------|
| 医薬品 | 注射薬、輸液、院内処方薬、迅速検査キット |
| 医療消耗品 | 注射針、シリンジ、スピッツ、手袋、ガーゼ、消毒綿 |
| 一般備品 | マスク、ペーパータオル、トイレットペーパー、ハンドソープ |
| 事務用品 | 印刷用紙（A4）、領収証・処方箋用紙（A5） |

## 在庫チェックのタイミング

- **毎日（終業時）**: 翌日の診療に支障がないか確認する。不足があれば院長に報告する。

> 開院日までに在庫管理チェックシートを用意します。

## 納品時の確認

1. 納品書と現物を照合する（品目・数量・ロット番号・使用期限）
2. 問題がなければ所定の場所に収納する
3. **使用期限の近いものを手前に配置する**（先入れ先出し）
4. 納品書をファイリングする

## 院内処方薬の管理

- **開封済みの処方薬から使う**
- 新たに箱を開封した場合は、**期限チェックシートの期限を更新する**
- 開封済みの薬がある状態で別の箱を開封すると期限がごっちゃになるため、**必ず確認を行う**
