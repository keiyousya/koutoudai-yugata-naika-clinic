# 緊急時対応

## 基本方針

患者の急変時は、**医師・看護師は患者対応に集中し、事務スタッフが救急要請を行う**。

## 救急要請（119番通報）

> 「救急です。仙台市青葉区国分町三丁目8-1、勾当台ビル4階の勾当台夕方内科クリニックです。患者様が突然○○となりました。」

- クリニックの電話番号: **022-393-8689**
- 通報後、エレベーター前まで救急隊を誘導する担当を決める

## アナフィラキシー

1. 原因物質の投与を直ちに中止する
2. 医師の指示でデカドロン iv / mv を投与する
3. バイタルサインを継続的にモニタリングする
4. 症状が改善しない場合は救急要請を行う

## CPA（心肺停止）

1. **CPR（胸骨圧迫）を開始する**
   - 強く（約5cm）、速く（100〜120回/分）、絶え間なく
   - 交代は2分ごとを目安にする
2. **AEDを装着する**
   - 音声ガイドに従い、ショック適応であれば実施する
3. **ECG肢誘導のみ装着**し波形を確認する
4. **末梢静脈路を確保**し生食を繋ぐ
   - **必ず三方活栓を使用する**（薬剤投与に備える）
5. 医師の指示で心筋保護薬等の投与を行う
6. 救急隊到着まで絶え間なくCPRを継続する

## 役割分担の目安

| 役割 | 担当 |
|-----|------|
| 患者対応・指示出し | 医師 |
| CPR・処置の補助 | 看護師 |
| 119番通報・救急隊の誘導 | 事務スタッフ |
| 記録（時系列メモ） | 手の空いたスタッフ |
