# 感染対策

## 診療時間帯の区分

| 時間帯 | 対象 |
|-------|------|
| 17:00 - 18:45 | 非発熱患者 |
| 19:00 - 21:00 | 発熱患者 |

- 非発熱の時間帯に発熱患者が受診する場合は**診察室2**で対応する
- 診察室2を使用した後は、次の患者を案内する前に環境消毒を行う

## マスク着用

- 院内は**マスク着用必須**とする
- 院内掲示を入口・受付・待合に設置する
- マスクを忘れた方には受付でマスクをお渡しする

## 手指衛生

- 患者対応の前後に手指消毒を行う
- 受付・待合・各診察室にアルコール消毒液を設置する
- 患者にも来院時の手指消毒を促す

## 検体の取り扱い

- 採血・尿検査の検体は素手で触れず、必ず手袋を着用する
- 使用済みの針は針捨てボックスに廃棄する（リキャップ禁止）
- 血液・体液が付着した場合は直ちにアルコールで清拭する

## 感染性廃棄物の処理

- 血液・体液が付着したガーゼ・手袋等 → **感染ゴミ**
- 使用済み注射針・メス等 → **針捨てボックス**
- 尿カップ → 小ゴミ袋で封をして感染ゴミへ
