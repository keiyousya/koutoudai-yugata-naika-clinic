2026-07-16〜2026-07-31 はクリニック休診。これに合わせて Google 広告の稼働キャンペーン
「2026年06月リスティング広告」（唯一の ENABLED / 日予算1,800円）に配信終了日時
`2026-07-15 23:59:59` を設定済み（2026-07-14 実施）。7/16以降は自動的に配信停止する。

**8/1（土）に手動で再開が必要**:

```
gads campaign end-date --campaign-id <CAMPAIGN_ID> --clear
```

解除が効くことは実機で検証済み（設定→解除→再設定を通した）。

**Why:** 休診中に広告を出すと来院できない患者のクリックに費用が出るだけで無駄。ただし
7/15（水）は通常診療日なので、即PAUSEDにせず終了日方式にして7/15分の配信は残した。

**How to apply:** 8月に入ったら真っ先に `--clear` で再開する。再開を忘れると休診明けも
広告が止まったままになる。求人・アフターピルの2キャンペーンは REMOVED 済みで操作不要
（[[afterpill-google-ads-not-approvable]]）。配信時間帯は [[clinic-ad-schedule-ops]] を参照。
