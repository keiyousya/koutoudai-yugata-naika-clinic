# 人事・労務

## 勤務時間

| 項目 | 内容 |
|-----|------|
| 診療時間 | 17:00 - 21:00 |
| 診療日 | 水・木・金・土・日 |
| 休診日 | 月・火・祝日 |

- 出勤は診療開始の15分前を目安とする（準備・申し送り）
- 終業は診察室・受付の片付け・清掃完了後とする

## 身だしなみ

### 基本方針

患者に清潔感と安心感を与える身だしなみを心がける。

### 規定

| 項目 | 内容 |
|-----|------|
| 服装 | 所定のユニフォーム（スクラブ等）を着用する |
| 髪 | 清潔に整え、長い場合はまとめる。業務に支障のない範囲で |
| 爪 | 短く切りそろえる。ネイル・付け爪は禁止 |
| アクセサリー | 業務中の指輪・ブレスレット・長いネックレスは外す（感染対策・安全のため） |
| 香水 | 患者が体調不良であるため、強い香りは避ける |
| 靴 | 院内用の清潔なシューズを使用する |

## 休暇

- 休暇の申請は可能な限り早めに院長へ相談する
- シフト調整が必要なため、希望日の2週間前までを目安とする

## 守秘義務

- 患者の個人情報・診療内容は院外で話さない
- SNS等への患者情報の投稿は厳禁
- 業務上知り得た情報は退職後も秘密を保持する
