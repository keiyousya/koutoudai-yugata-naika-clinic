# 緊急時対応

## 基本方針

患者の急変時は、医師・看護師は患者対応に集中し、事務スタッフが救急要請を行う。
> 救急要請は医業ではないため、事務スタッフが対応して問題ありません

## 緊急医薬品の保管場所

- 緊急医薬品は**スタッフルームの調剤スペースにケースでまとめている**
- 急変時は**すぐに持ってくる**
- **デカドロンのみ冷蔵庫に保管**しているため、使用する際は冷蔵庫から持ってくる

## 救急要請（119番通報）

> 「救急です。仙台市青葉区国分町三丁目8-1、勾当台ビル4階の勾当台夕方内科クリニックです。患者様が突然○○となりました。」

- クリニックの電話番号: **022-393-8689**
- **患者の保険証と直近のカルテ内容を印刷し用意する**
- 通報後、**エレベーターを1階に下ろしておく**
- 事務スタッフは**1階エントランスで待機し、救急隊を誘導する**

## アナフィラキシー

1. 原因物質の投与を直ちに中止する
2. 患者を適切な体位にする
   - 仰臥位
   - 呼吸困難がある場合は座位
   - 意識消失状態なら回復体位
3. 医師、または医師の指示で看護師が、アドレナリンを筋肉注射する
   - （成人）アドレナリン0.5mg im
   - （小児）アドレナリン(0.01 x 体重)mg im
   - 投与時刻を記録し、5-15分毎に再投与
4. 末梢確保して生食を繋ぐ（可能であれば20G）
5. バイタルサインを継続的にモニタリングする
6. 症状が改善しない場合は救急要請を行う

## CPA（心肺停止）

1. **CPR（胸骨圧迫）を開始する**
   - 強く（約5cm）、速く（100〜120回/分）、絶え間なく
   - 交代は2分ごとを目安にする
2. **AEDを装着する**
   - 音声ガイドに従い、ショック適応であれば実施する
3. **ECG肢誘導のみ装着**し波形を確認する
4. **末梢静脈路を確保**し生食を繋ぐ
   - **必ず三方活栓を使用する**（薬剤投与に備える）
5. 医師の指示で心筋保護薬等の投与を行う
6. 救急隊到着まで絶え間なくCPRを継続する

## 役割分担の目安

| 役割 | 担当 |
|-----|------|
| 患者対応・指示出し | 医師 |
| CPR・処置の補助 | 看護師 |
| 119番通報・救急隊の誘導 | 事務スタッフ |
| 記録（投与した薬剤や処置行為を時系列に記録） | 手の空いたスタッフ |
