# 会計業務

## 基本の流れ

診察終了後、ORCA（レセコン）で請求金額をお伝えし、会計方法を確認する。

## 現金決済

1. 患者に請求金額をお伝えする
2. 現金を受け取り、お釣りがあればお返しする
3. ORCAで登録を行う
4. **領収証明細書**を発行しお渡しする

## 対面キャッシュレス決済

1. 患者に請求金額をお伝えする
2. キャッシュレス端末で支払いを完了する
3. ORCAで登録を行う
4. **領収証明細書**を発行する
5. キャッシュレス端末から発行される**売上票**と合わせてお渡しする

## LINEクレジットカード決済

1. カルテから「LINEクレカ決済」を実行する
2. 支払い完了を確認する
3. ORCAで登録を行う
4. **領収証明細書**をお渡しする

> **注意**: エラーで支払いが完了しない場合は、対面での支払い（現金またはキャッシュレス）をお願いする。

## 処方箋のお渡し

- 処方箋がある場合は会計時にお渡しする
- 当院は原則**院外処方**（近隣の調剤薬局で受け取り）
- 近隣薬局が閉まっている時間帯に限り、急性疾患の一部薬剤は院内でお渡しする場合がある
- 慢性疾患の薬（降圧薬・糖尿病治療薬等）は必ず院外処方とする
