# 衛生管理（院内清掃）

## トイレチェック

汚れがあれば清掃し、**チェックシートに名前と時間を記入する**。

| 時間 | 内容 |
|------|------|
| 18:00 | チェック（汚れがあれば清掃） |
| 19:00 | チェック（汚れがあれば清掃） |
| 20:00 | チェック（汚れがあれば清掃） |
| 21:00 | **必ず清掃を行う** |

### 清掃内容

- 便器・便座の拭き掃除
- 床の汚れの確認・清拭
- トイレットペーパー・ハンドソープの補充
- ゴミ箱の確認（溜まっていれば交換）

## 待合・受付の環境整備

- 手の空いたタイミングで**待合・受付のモップがけ**を行う
- 椅子・テーブルの消毒（特に多くの患者が触れる箇所）
- パンフレット類の整頓

## 診察室の清掃

- 患者ごとに診察台のロールシーツを交換する
- 診察台・デスク周りをアルコールで清拭する
- 発熱患者の診察後は特に入念に環境消毒を行う

## 終業時の清掃

> **21時前までに以下を終了する**                                                                   
> - 終わらなければ**LINE WORKSのノート**で共有する
> - 引き継ぎ項目は、翌日勤務者が出勤後に実施する

1. 待合室・受付のモップがけ
2. トイレの最終清掃（21:00のチェック）
3. ゴミの回収・分別
4. 翌日の備品（ペーパータオル・消毒液等）の補充確認
