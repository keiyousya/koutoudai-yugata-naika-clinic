有効キャンペーンは「2026年06月リスティング広告」1本（campaign_id `<CAMPAIGN_ID>` / ad_group `<AD_GROUP_ID>`）。手動CPC・日予算約2,000円。

**診療時間（2026-07-04にスクショで確認した正式版）:**
- 月・水・木・金: **17:00–21:00**（夕方のみ。午前・日中は診療なし）
- 火: **休診**
- 土・日: **14:00–21:00**
- 休診日: 火曜・祝日／**最終受付 20:45**

**広告配信の現状と実態のズレ（要調整）:**
前回(2026-07-04)の配信変更は「平日は12-20時で14-16時が昼休み」という誤った前提で設定してしまった。実際は平日=17-21時のみ・土日=14-21時。現在アカウントに入っている配信は:
- 平日(月水木金): 12-13,13-14,16-17,17-18,18-20時（12時×1.20/17時×1.25）→ **診療時間17-21に対しズレている（開院前の時間に多く配信・21時前の20時で終了）**
- 土日: 12:00-20:00 据え置き → 診療14-21に対しズレ
→ 診療時間に合わせて再設計するかは要相談。

**重要な運用インサイト（オーナー談・2026-07-04）:** 平日は診療が17-21時なのに **12-14時台のCVが多い**。これは休診時間の無駄ではなく、**社会人が昼休憩にスマホで探して夜の受診を予約する「先取り検索」**。よって昼(12-14時)の配信はCVに効くので絶対に残す。診療時間に配信を寄せる場合も、昼帯はむしろ強化方向で扱い、一律に診療時間内へ絞らないこと。終了は最終受付20:45に合わせ21:00まで伸ばすのが理にかなう。

**How to apply:** 配信時間/入札の変更はCLI `gads campaign schedule` が入札調整・既存削除に非対応のため、`gads.client` を使う専用スクリプト（remove既存→create新規、bid_modifier指定）で行う。除外キーワードは `gads keyword add --negative` で可。他院の指名検索（厚生病院/大学病院/ジェイコー/徳洲会/上杉病院/みやざき内科/はじめのクリニック）は除外済み。アフターピル広告は不可（[[afterpill-google-ads-not-approvable]]）。
