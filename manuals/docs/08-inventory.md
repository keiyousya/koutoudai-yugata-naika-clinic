# 在庫管理

## 管理対象

| 分類 | 例 |
|-----|------|
| 医薬品 | 注射薬、輸液、院内処方薬、迅速検査キット |
| 医療消耗品 | 注射針、シリンジ、スピッツ、手袋、ガーゼ、消毒綿 |
| 一般備品 | マスク、ペーパータオル、トイレットペーパー、ハンドソープ |
| 事務用品 | 印刷用紙、領収証用紙、処方箋用紙 |

## 在庫チェックのタイミング

- **毎日（終業時）**: 翌日の診療に支障がないか確認する。不足があれば翌朝までに発注する
- **毎週（決まった曜日）**: 全品目の在庫数を確認し、発注基準を下回っているものを発注する

## 発注の流れ

1. 在庫チェックリストで残数を確認する
2. 発注基準を下回っている品目をリストアップする
3. 所定の発注先に連絡する（発注先リストは別途管理）
4. 発注内容を記録に残す

## 納品時の確認

1. 納品書と現物を照合する（品目・数量・ロット番号・使用期限）
2. 問題がなければ所定の場所に収納する
3. **使用期限の近いものを手前に配置する**（先入れ先出し）
4. 納品書をファイリングする

## 医薬品の温度管理

| 保管条件 | 対象 |
|---------|------|
| 常温保管 | 一般的な内服薬・外用薬 |
| 冷蔵保管（2〜8℃） | 冷蔵指定の注射薬・検査試薬 |
| 冷凍保管 | BNP・凝固検体（遠心分離後の血漿） |

- 冷蔵庫・冷凍庫の温度を毎日確認し、記録する
