# 人事・労務

|-----|------|

## 希望シフト・休みについて

- シフト変更・休みの希望は可能な限り早めに院長へ相談する
- シフト調整が必要なため、希望日の2週間前までを目安とする

## 身だしなみ

### 基本方針

患者に清潔感と安心感を与える身だしなみを心がける。

### 規定

| 項目 | 内容 |
|-----|------|
| 服装 | 所定のユニフォーム（スクラブ等）を着用する |
| 髪 | 清潔に整え、長い場合はまとめる。業務に支障のない範囲で |
| 爪 | 短く切りそろえる。ネイルは長すぎず、派手すぎず、清潔感のあるものであればOK |
| アクセサリー | 業務中の指輪・ブレスレットは外す（感染対策・安全のため） |
| 香水 | 強い香りは避ける |
| 靴 | 院内用の清潔なシューズを使用する |

## 守秘義務

- 患者の個人情報・診療内容は院外で話さない
- SNS等への患者情報の投稿は厳禁
- 業務上知り得た情報は退職後も秘密を保持する
